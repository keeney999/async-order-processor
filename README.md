# Async Order Processor

Система асинхронной обработки заказов с использованием очередей (RabbitMQ + Celery) и продвинутого SQL.

## Стек технологий

- **FastAPI** – асинхронный веб-фреймворк.
- **SQLModel** – ORM на основе SQLAlchemy и Pydantic.
- **PostgreSQL** – основная база данных.
- **RabbitMQ** – брокер сообщений (бэкенд Celery).
- **Redis** – хранилище результатов Celery.
- **Celery** – распределённая очередь задач.
- **Alembic** – миграции базы данных.
- **Docker** – контейнеризация.
- **Pytest + testcontainers** – тестирование.

## Функциональность

- API для создания заказа с атомарной записью в outbox.
- Фоновая обработка заказа (резервирование товаров с использованием сложного SQL).
- Автоматическая отправка событий в RabbitMQ через outbox.
- Оплата заказа (имитация) с обновлением статуса.
- Продвинутые SQL-запросы: оконные функции для статистики, массовое обновление с проверкой остатков.
